HTML Study Notes

HTML's full name is HyperText Markup Language

Basic Frame

	< !DOCTYPE html > 

	<html lang="en" > 

		<head > 

			<meta charset="UTF-8" > 

			<title >< /title > 

		</head > 

		<body > 

		</body > 

	</html >

Some target in common use

Framework

  • html : tell browser this is a html file. It define the starting point and end point.
  • head : define the document's heading, it is the container of all heading elements. In <head> element, we can quote scripts, instruct the browser where to find style sheets, provide the meta-information. < head > describes the document properties and information, contain the document title,the location in Web and the relationship with other docuements. Most data in < head > should not appears in web to readers. Following label can be use in head: base,link,meta,script,style title. The label of title is necessary for head, it shows in the web's label, not in the content of web.
  • body : define the body of the documents. The content in < body > will present to the reader. In <body>, we can use background to set the background of the web page. If the picture is smaller than the page, it will repeated.

Body

  • header : define the header of the web page.
  • footer : define the footer of the web page or section. A document can have some <footer>.
  • Paregraphy
    • h1 &minus h6 : define the title. h1 define the largest title, h6 define the smallest title.
    • p : define the paragraph.
    • br : insert a simple newline. <br> has no end tag. When browser meet <p>, generally insert a simple newline between two adjacent paragraphs.
    • Insert
    • img : Insert picture. In fact <img> don't insert a picture in web page, just link to pictures. It has two nessary elements: src and alt.
      <img src="picture link" alt="name"/>
      property :
      • src : the URL of the picture
      • alt : provides an alternative text of the image
      • height : the height of picture (pixels)
      • width : the width of picture (pixels)
    • table : define the table. <th> define meter unit; <td> define standard unit. <tr> define the line in table. 
      					<table border="1">
						<tr>
							<th>Month</th>
							<th>Savings</th>
						</tr>
						<tr>
							<td>January</td>
							<td>$100</td>
						</tr>
					</table>
      Month Savings
      January $100
      property :
      • align : define the contents of a table row alignment (right,left,center,justify,char).
      • valign : define the contents of a table column alignment (top, middle,bottom,baseline).
      • char : depending on which character requirements for text alignment.
      • charoff : provision the offset alignment of the first character.
